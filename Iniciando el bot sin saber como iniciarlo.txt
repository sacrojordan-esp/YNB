1 Descargar e instalar Python entrando a https://www.python.org
2 Descargar el ZIP y extraer en el escritorio, renombra la carpeta y llamala YNB
3 Accede al archivo .env y agrega tu token entre las comillas: DISCORD_TOKEN='tu_token_aqu√≠'

---Instalacion del bot---
En el buscador de Windows escribir CMD y dar ENTER - Esto abrira "Simbolo del Sistema"
Pegar cada uno de estos codigos y dar ENTER.

cd %userprofile%\Desktop\YNB       Navegar al Escritorio		
py -m venv venv                    Crear un entorno virtual
.\venv\Scripts\activate            Activar el entorno virtual	
pip install -r requirements.txt    Instalar requirements.txt

-Cerrar la ventana-

---Para ejecutar el bot---
En el buscador de Windows escribir CMD y dar ENTER - Esto abrira "Simbolo del Sistema"
Pegar cada uno de estos codigos y dar ENTER.

cd %userprofile%\Desktop\YNB       Entrar a la carpeta
.\venv\Scripts\activate            Activar el entorno virtual
python main.py	                   Ejecutar el bot

CONTROL + C para detener el proceso del bot ejecutandose
